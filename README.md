# Sentiment-Analysis-on-Tourist-Reviews-in-India
NLP-based sentiment analysis system that classifies 30K+ tourist reviews into positive, neutral, and negative categories using TF-IDF and SVM (~92% accuracy), with a recommendation component to identify top tourist destinations based on sentiment trends.


## 📖 Project Overview
This project focuses on building an intelligent **Sentiment Analysis System** to analyze tourist reviews from various Indian destinations. Using **Natural Language Processing (NLP)** and **Machine Learning**, the system classifies reviews into **Positive, Neutral, and Negative sentiments**.

It also includes a **recommendation system** that suggests destinations based on overall sentiment trends, helping users make better travel decisions.

---

## 🎯 Objectives
- Classify tourist reviews into positive, neutral, and negative sentiments  
- Apply NLP techniques for text preprocessing and analysis  
- Build an end-to-end machine learning pipeline  
- Use TF-IDF for feature extraction  
- Evaluate models using standard metrics  
- Provide sentiment-based recommendations  

---

## 🛠️ Tech Stack
- **Language:** Python  
- **Libraries:** Pandas, NumPy, Scikit-learn, NLTK, Matplotlib, Seaborn  
- **Techniques:** NLP, TF-IDF, Machine Learning  

---

## ⚙️ Methodology
1. Data Collection (tourist reviews dataset)  
2. Data Preprocessing (tokenization, stopword removal, lemmatization)  
3. Sentiment Labeling (based on ratings)  
4. Feature Extraction (TF-IDF)  
5. Model Building (Logistic Regression, SVM)  
6. Model Evaluation (Accuracy, Precision, Recall, F1-score)  
7. Visualization (word clouds, graphs)  

---

## 📊 Results
- Achieved ~92% accuracy using SVM  
- Successfully classified 30K+ reviews  
- Identified sentiment trends across destinations
